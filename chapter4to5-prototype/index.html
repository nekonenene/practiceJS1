2: 学んだこと

prototype では this. で変数を指定するものだと思っていたけど、
それはオブジェクトに関連付けられてしまうので、
オブジェクトのコンストラクタ以外では変数の宣言に使っちゃあかん、ってわかった。

どうなるかって言うと、this.value なんかメソッド内で宣言した日にゃ、
その this.value は全体のメソッドで共通になるので、
他のメソッドで this.value と仮に宣言したら、同じものを使いまわすことになる。危険。

また、 function も変数だから、
this.sumDigits で数値を入れる変数を宣言したつもりが、
オブジェクトのメソッド function sumDigits() と誤認識されて、わけわからないことになる。
